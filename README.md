# Geospatial Data Transformation & Map Sheet Design (Netcad)
This project focuses on coordinate system transformations and raster-to-vector alignment using Netcad.

## Key Operations
- **Georeferencing:** Performed raster transformation for the F20-C (Silivri) map sheet.
- **Coordinate Transformation:** Executed datum transformation from WGS84 to TM30 (ED50/ITRF) standards.
- **Cartographic Design:** Prepared progress and reconnaissance sketches for the project area.
